# AutomationExercise Playwright Project

Playwright tests for the AutomationExercise ecommerce demo site.

## Run

```bash
npm install
npx playwright install
npm test
```

## Project Structure

- `Automationexercise/automationexercise-ecommerce.spec.ts` - main ecommerce journey test
- `Automationexercise/pages/` - page objects
- `Automationexercise/utils/testData.ts` - generated test data helpers
- `playwright.config.ts` - Playwright configuration
